\abstractCS{Náplňou tejto bakalárskej práce je analýza súčasnej webovej aplikácie používanej pre evidenciu zvierat, vrhov a ich odchovov neziskovou organizáciou \uv{Český klub potkanů}. Na základe analýzy je navrhnutá a implementovaná webová SPA aplikácia s optimálnejšou štruktúrou databázy, ktorá okrem iného rieši duplicitné ukladanie dát. Správnosť funkcionality aplikácie je následne overená integračnými testami.}
\abstractEN{The content of this bachelor thesis is the analysis of the current web application used for the registration of animals, litters and their breeding by the non-profit organisation \uv{Czech Fancy Rat Club}. Based on the analysis, a web SPA application with a more optimal database structure is designed and implemented, which, among other things, solves duplicate data storage. The correctness of the application functionality is subsequently verified by integration tests.}
